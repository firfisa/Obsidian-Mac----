# Widget 交互功能配置

## 功能说明

新的 widget 支持以下交互功能：

1. **点击复选框完成任务** - 点击任务前的 ☐/✓ 可以切换任务状态
2. **添加新任务** - 点击右上角的 "+ 添加" 按钮，输入任务内容后按回车

## 配置步骤

### 1. 启用交互功能

Übersicht 的交互功能需要配置快捷键：

1. 打开 Übersicht 应用
2. 点击菜单栏的 Übersicht 图标
3. 选择 "Preferences" 或 "设置"
4. 找到 "Interaction Shortcut" 或 "交互快捷键"
5. 设置一个快捷键（例如：`Cmd + Shift + I`）
6. 确保 Übersicht 有辅助功能权限：
   - 系统设置 > 隐私与安全性 > 辅助功能
   - 确保 Übersicht 已勾选

### 2. 使用交互功能

**完成任务：**
- 点击任务前的 ☐ 或 ✓ 复选框
- Widget 会自动更新文件并刷新显示

**添加任务：**
- 点击右上角的 "+ 添加" 按钮
- 输入任务内容
- 按回车键确认
- 任务会自动添加到文件顶部

## 故障排除

**点击没有反应：**
1. 检查是否配置了交互快捷键
2. 检查 Übersicht 是否有辅助功能权限
3. 查看 Übersicht 日志（菜单栏 > Übersicht > Logs）

**添加任务失败：**
- 检查 `todo_helper.py` 脚本是否有执行权限
- 检查文件路径是否正确
- 查看终端错误信息

## 技术说明

交互功能通过以下方式实现：
- 使用 Übersicht 的 `run` 函数执行 Python 脚本
- `todo_helper.py` 脚本处理文件修改
- Widget 自动刷新显示最新内容
